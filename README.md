# pn-emd-integration

Project that integrates with the API exposed by EMD's services to provide the ability to send messages to third-party channels.

## Istruzioni per la compilazione
```
    ./mvnw clean install
```

## Run locale
Avvio cache redis
```
    docker compose up
```

Avvio applicazione
```
    ./mvnw spring-boot:run
```
